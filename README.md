🎁 Challenge Amigo Secreto - Oracle ONE

¡Bienvenido al desafío del Amigo Secreto! 🎉

¿Alguna vez has tenido problemas organizando un intercambio de regalos? Con esta aplicación web, puedes ingresar nombres y obtener aleatoriamente quién será el afortunado receptor del regalo. ¡Deja que el destino decida por ti!

🚀 Tecnologías utilizadas

🌐 HTML: Para estructurar la página.

🎨 CSS: Para darle estilo y hacerla visualmente atractiva.

⚡ JavaScript: Para la lógica de selección aleatoria.

🎲 ¿Cómo funciona?

Ingresa los nombres de los participantes.

Haz clic en el botón para asignar un nombre al azar.

¡Descubre quién es el elegido!


💡 ¿Por qué este proyecto?
Este reto forma parte del programa Oracle ONE.
